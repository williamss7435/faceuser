## Instalação
### Back-end 
```
Criar Tabela faceuser no banco de dados
cd backend-faceuser
npx sequelize-cli db:migrate (Carregar schema das tabelas)
npm run dev
```

### Front-end (Web)
```
cd web-faceuser
npm install
npm start
```
